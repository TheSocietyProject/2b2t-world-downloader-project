#WorldDownloader tutorial
-start minecraft 1.12.2

-download forge 1.12.2 at https://files.minecraftforge.net/
-execute and install the client
-start Minecraft with forge

-download liteloader (needed for worlddownloader) at:
https://www.liteloader.com/download#snapshot_11220 -> 1.12 exe or jar
-open and select: extends from: 1.12.2-forge1.12.2-14.23.5.2772 and press ok

-download WorldDownloader at: 
https://github.com/Pokechu22/WorldDownloader/releases/
-> mod-litemodwdl-4.0.5.3-mc1.12.2.litemod

-if you want to use hacks together with worldownloader I recommend kami:
-> https://github.com/zeroeightysix/KAMI/releases / KAMI-b9-release.jar

-move all the mods in your .minecraft folder (you get there by pressing windows + R -> %appdata% -> .minecraft/mods)

In MC:
-when starting download server for the first time, its best to select Disable Multiworld
because otherwise you have to select a world where you are in when starting a download and when switching dimensions.
